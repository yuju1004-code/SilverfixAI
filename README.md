# 디지털 손주 SilverFix AI (실버픽스 AI) MVP

어르신(실버 세대)들을 위한 스마트폰 도우미 및 감성 동반자, 디지털 손주 **SilverFix AI** 프로토타입입니다.

본 프로젝트는 고령층이 일상생활에서 겪는 디지털 소외 문제를 해결하고 스마트폰 적응력을 키우며, 말벗 역할을 수행하는 것을 목표로 합니다.

## 주요 기능

1. **👦/👧 손주와 대화하기 (AI Chat)**
   - 한국어 음성 인식(STT) 및 음성 합성(TTS) 지원으로 타자 입력 없이 편리한 대화 가능.
   - 와이파이, 카톡 사진 전송, 플레이스토어 등 자주 묻는 디지털 기기 관련 질문에 대해 친근한 비유로 설명.
   - **실시간 AI 연동 지원**: 설정 화면에서 OpenAI 또는 Google Gemini API Key를 등록하면 모의 답변이 아닌 실제 AI와 고도화된 자유 대화 가능.

2. **📖 스마트폰 낱말사전 (Glossary)**
   - 와이파이, 블루투스, QR코드, 키오스크 등 낯설고 어려운 정보통신 용어들을 어르신 눈높이에 맞는 비유와 꿀팁으로 설명.
   - 카테고리 필터링 및 쉬운 단어 검색 지원. 전체 설명을 목소리로 읽어주는 기능 탑재.

3. **🎮 스마트폰 연습실 (Simulator)**
   - 가상의 스마트폰 화면을 통해 안전한 환경에서 실습 수행.
   - **키오스크로 햄버거 주문하기**, **카카오T로 택시 부르기**, **유튜브로 임영웅 노래 검색하기** 시뮬레이션 탑재.
   - 진행 단계별 하이라이트(Highlight) 및 손주 가이드 음성 안내 제공.

4. **✍️ 목소리 일기장 (Voice Diary)**
   - 오늘 하루 동안 일어난 일을 말하면 자동으로 텍스트 변환하여 기록.
   - 일기 텍스트의 분위기를 감성 분석하여 자동으로 기분 이모티콘과 테마가 지정된 폴라로이드 사진 일기 생성.
   - 손주 캐릭터의 다정한 한마디 편지 동봉.
   - **폴라로이드 일기장 저장**: 캔버스 드로잉을 통해 폰트와 손글씨 느낌이 살아있는 이미지(PNG) 파일로 다운로드 가능.

5. **⚙️ 어르신 맞춤형 편의 설정**
   - **글씨 크기 및 고대비 모드**: 눈이 어두우신 어르신들을 위해 글자 크기를 키우고 명도 대비를 최대로 높인 고대비 모드 지원.
   - **목소리 속도 조절**: 듣기 편한 목소리 속도(0.6배속 ~ 1.4배속)로 오디오 출력 설정 가능.

## 설치 및 실행 방법

### 로컬 실행 (Vite 개발 서버 이용)

1. 저장소를 클론(Clone)하거나 다운로드 받습니다.
2. 터미널에서 필요한 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. 로컬 개발 서버를 실행합니다.
   ```bash
   npm run dev
   ```
4. 터미널에 나타난 로컬 주소(`http://localhost:5173`)로 접속합니다.
